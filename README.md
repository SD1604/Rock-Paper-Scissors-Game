# 🪨 📄 ✂️ Rock Paper Scissors

A classic Rock Paper Scissors game against a computer opponent.

[Play the game here!](PASTE_YOUR_LINK_HERE)

## 🎮 How to Play

1. Select your weapon (Rock, Paper, or Scissors).
2. The computer will simultaneously make a random choice.
3. The winner is determined based on:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock

## 🧠 Logic Highlights

- **Randomized AI:** Uses `Math.random()` to generate computer moves.
- **Score Tracking:** Local state keeps track of User vs. Computer wins during the session.
- **DOM Manipulation:** Real-time UI updates for results and animations.

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)
